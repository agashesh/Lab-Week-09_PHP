# PHP Basic - Tugas Praktikum Pemrograman Web

Proyek ini dibuat untuk memenuhi tugas praktikum PHP Basic.

## File Program

- `biodata.php` → Menampilkan biodata mahasiswa menggunakan variabel.
- `kalkulator_nilai.php` → Menghitung nilai akhir, grade, dan status kelulusan.
- `daftar_produk.php` → Menampilkan daftar produk menggunakan array multidimensional dan function format rupiah.

## Cara Menjalankan

1. Jalankan Apache pada XAMPP.
2. Simpan folder project di:

```text
C:\xampp\htdocs\php_basic
```

3. Buka browser dan akses:

```text
http://localhost/php_basic/biodata.php
http://localhost/php_basic/kalkulator_nilai.php
http://localhost/php_basic/daftar_produk.php
```
